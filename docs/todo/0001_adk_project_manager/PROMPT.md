Install a project manager agent using ADK typescript running on cloudflare agents sdk and wire  it up as an assistant-ui modal chat on the project management pages on the frontend (projects, tasks, kanban, etc). 

You can retrofit from this project manager agent: `https://github.com/Sri-Krishna-V/awesome-adk-agents/tree/main/my-adk-agents/project-manager-agent` as well as from this example: `https://github.com/google/adk-samples/tree/main/python/agents/sdlc-task-planner`.
